SHOPIFY INTEGRATION
===================

The sitemap and product pages connect to Shopify via:

Environment Variables (already configured):
- SHOPIFY_STORE_DOMAIN
- SHOPIFY_STOREFRONT_ACCESS_TOKEN

Used By:
- D:\gymsnacks-custom\gymsnacks-v2\src\lib\shopify.ts (unchanged)

Fetches:
- All products from Shopify Storefront API
- Product handles, titles, descriptions, prices, images
- Availability status

Auto-Updates:
- Sitemap regenerates on each request
- Product pages regenerate when products change in Shopify
