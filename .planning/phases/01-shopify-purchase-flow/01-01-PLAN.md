**Known concern:** `getProductByHandle` uses string interpolation in GraphQL (`handle: "${handle}"`) instead of variables. This works but is not best practice. If handle contains special characters it could break. Fix by using GraphQL variables:
```
query GetProduct($handle: String!) {
  productByHandle(handle: $handle) { ... }
}
```
And pass `variables: { handle }`.
